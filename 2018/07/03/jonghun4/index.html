AWS Cognito를 위한 단어집-1

쓴 날: 2018/7/3

람다 서비스를 이용할 수 있는 유저를 제한하려고 한다.
Facebook과 연동해 가입한 사람만 이용가능하게 하려 한다.
AWS Cognito를 이용해 이 작업을 하려고 한다.
문서를 읽다보니 생소한 용어가 많아 조금씩 정리한다.

Voca Summary Stands for
SAML[1] 데이터 포맷, 개방형 표준이며 XML 기반.
IdP와 SP 간에 인증 및 인가 데이터를 교환하기 위해 쓰임.		 Security Assertion Markup Language
IdP [2][3] 시스템, principals에 대한 identity 정보를 만들고, 유지하고, 관리.
유저와 서버에게 신뢰받는 써드 파티.		 Identity Provider
Principal [4][5] 컴퓨터 시스템에 의해 인증될 수 있는 양.
개인, 컴퓨터, 서비스, 프로세스와 쓰레드와 같은 계산적 량이거나,
그것들의 집합.
확인되고 인증된 후에 네트워크 자원에 접근할 수 있는 권한을 갖음.		 Security Principal이라고도 불림
SP [6] 시스템 단위의 역할,
이 역할의 시스템 단위는 다른 시스템 단위 또는 principals에게 서비스를 제공.		 Service Provider
SSO 권한 제어의속성, 서로 연관돼 있지만 독립적인 소프트웨어 시스템에서 사용.
이 속성이 있으면 하나의 ID와 패스워드 만으로 연결된 시스템 전체를 이용 가능.		 Single Sign-On
